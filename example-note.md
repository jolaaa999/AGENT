# 机器学习与线性代数速记

## 1. 监督学习基础

- 监督学习的目标是学习从输入 `x` 到输出 `y` 的映射函数。
- 常见任务包括分类（classification）与回归（regression）。
- 训练集用于拟合参数，验证集用于调参，测试集用于评估泛化能力。

## 2. 核心数学基础

### 2.1 向量与矩阵

- 向量可以表示样本特征，矩阵可以表示一批样本。
- 矩阵乘法是神经网络前向传播的核心运算。
- 内积可衡量两个向量方向一致性（相似度的一种形式）。

### 2.2 梯度与优化

- 梯度下降通过沿损失函数负梯度方向更新参数：

  `theta = theta - lr * grad`

- 学习率过大可能震荡，过小会导致收敛慢。
- SGD、Momentum、Adam 都是常见优化算法。

## 3. 模型评估

- 分类常见指标：Accuracy、Precision、Recall、F1。
- 类别不平衡时，Accuracy 可能具有误导性。
- ROC-AUC 可用于衡量模型区分正负样本的能力。

## 4. 易错点与补充

- 错误认知：只要训练集准确率高，模型就一定好。  
  纠正：这可能是过拟合，应关注验证集和测试集表现。

- 错误认知：特征越多越好。  
  纠正：无关特征会引入噪声，可能降低泛化能力。

- 需要补充：偏差-方差权衡（Bias-Variance Tradeoff）与正则化（L1/L2）。
- 需要补充：特征工程与数据清洗对模型效果的影响。

## 5. 跨学科联系

- 统计学提供了假设检验、分布建模与不确定性分析。
- 概率论支撑贝叶斯学习与马尔可夫模型。
- 信息论中的熵可用于衡量不确定性，也常用于决策树划分标准。
- 线性代数支撑降维方法，如 PCA。

## 6. 目标概念导航示例

如果目标概念是“深度学习”，建议前置路径可包含：

1. 线性代数 -> 梯度下降 -> 反向传播 -> 深度学习
2. 概率论 -> 损失函数 -> 优化算法 -> 深度学习
3. 模型评估 -> 过拟合 -> 正则化 -> 深度学习
